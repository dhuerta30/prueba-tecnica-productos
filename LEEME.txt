PRUEBA REGISTRO DE PRODUCTOS

Tecnologías utilizadas:
- PHP 8
- MySQL
- HTML5
- CSS3
- JavaScript (Fetch API)

Estructura:

/css
    style.css

/js
    app.js

/php
    conexion.php
    guardar_producto.php
    obtener_bodegas.php
    obtener_materiales.php
    obtener_monedas.php
    obtener_sucursales.php
    validar_codigo.php

/sql
    prueba_productos.sql

Instalación:

1. Importar el archivo sql/prueba_productos.sql en MySQL
2. Verificar los datos de conexión en php/conexion.php
3. Copiar el proyecto en el servidor web
4. Abrir:
   http://localhost/prueba-tecnica-productos

Funcionalidades implementadas:

- Carga de bodegas mediante AJAX.
- Carga de sucursales dependiente de la bodega seleccionada.
- Carga de monedas mediante AJAX.
- Carga de materiales mediante AJAX.
- Validación de código único.
- Validaciones de formulario en JavaScript.
- Guardado de productos y materiales asociados en MySQL.
- Uso de transacciones para asegurar integridad de datos.

Observaciones:

La solución fue desarrollada utilizando JavaScript nativo (Fetch API) y PHP con PDO para acceso seguro a la base de datos.
